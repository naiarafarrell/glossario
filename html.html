Tags HTML

Links

Você pode clicar em um link e pular para outro documento.

<a href="https://www.w3schools.com/">Visit W3Schools.com!</a>

Título

O elemento <title> adiciona um título à sua página.

<title>HTML Tutorial</title>

Cabeçalhos HTML

Títulos HTML são títulos ou subtítulos que você deseja exibir em uma página da web.

<h1>Este é um título</h1>

Strong

Indica que o conteúdo tem importância forte, geralmente apresentado em negrito.

<p>Este produto está em <strong>promoção</strong> hoje!</p>

Button

Cria um botão clicável na página, que pode executar ações com JavaScript ou enviar formulários.

<button>Enviar</button>

A

Tag para criar links clicáveis. O atributo href indica para onde o link vai.

<a href="https://exemplo.com">Clique aqui</a>

Main

Identifica o conteúdo principal da página. Deve haver apenas um <main> por página.

<main>
<h1>Olá, mundo!</h1>
<p>Conteúdo principal aqui.</p>
</main>

Header

Define o cabeçalho de uma página ou seção. Normalmente inclui títulos, logotipos ou navegação.

<header>
<h1>Meu Site</h1>
</header>

Nav

Indica uma área de navegação no site, geralmente contendo links para as principais páginas.

<nav>
<a href="#">Início</a>
<a href="#">Sobre</a>
<a href="#">Contato</a>
</nav>

Footer

Marca o rodapé da página. Costuma conter direitos autorais, links ou contatos.

<footer>
<p>© 2025 Meu Site. Todos os direitos reservados.</p>
</footer>

Propriedades do CSS

Color

Define a cor do texto.

p { color: red; }

Background-color

Define a cor de fundo de um elemento.

div { background-color: yellow; }

Font-size

Define o tamanho da fonte.

h1 { font-size: 24px; }

Text-align

Alinha o texto dentro de um elemento.

p { text-align: center; }

Margin

Define a margem externa do elemento.

div { margin: 10px; }

Padding

Define o espaço interno entre o conteúdo e a borda.

div { padding: 5px; }

Width

Define o tamanho da largura

Width: 100%;

Border

Define a borda do elemento.

div { border: 1px solid black; }

Height

Define a altura de um elemento.

Height: 100%;

Border-radius

Define o arredondamento da borda.

border-radius: 12px;
